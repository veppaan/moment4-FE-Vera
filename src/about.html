Moment 1 - Frontend-baserad webbutveckling

Vera Kippel

Arbetsprocessen

Det började med att jag öppnade terminalen för att skapa ett NodeJs projekt med kommandot "npm install". Efter det så laddade jag ner paketet "parcel".

När det var skapat så kopplade jag ihop projektet med github för att kunna göra commits under projektets gång.


Sidan skulle innehålla tre undersidor, så en efter en skapades med hjälp av HTML, CSS och JavaScript. Menyraden skapades allra först där den har en synlig menyrad i stor skärm och sedan en hamburgermeny i tablet-och mobil-skärm. Hamburgermenyns funktion skapades med hjälp av JavaScript.

Efter att menyraden hade skapats så lade jag in allt innehåll i sidorna samtidigt som designen uppdaterades med CSS för att allt skulle se fint och strukturerat ut.

Bilderna lades in i picture-tagg för att kunna ha olika filformat som optimerar sidan. Jag använde mig av avif, sedan webp och sist jpeg som fallback (som är bildernas original-format).


För att webbplatsen ska kunna användas av så många webbläsare som möjligt så lade jag in "browserlist" som gör att mina filer konverteras efter användarens webbläsare.


När allt innehåll och design lagts in och jag var nöjd med resultatet så gick projektet över till Github och Netlify.

Väl på github så kollade jag att mina filer i gitignore inte inkluderats. Efter det så gick jag till Netlify där jag publicerade mitt projekt genom github där jag ställde in att sidan skulle köra kommandot "npm run build" som optimerar och kompilerar webbplatsen och gör allt innehåll färdigt för publicering.
